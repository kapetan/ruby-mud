Ruby port of Mud (package manager for client-side Javascript). See [http://github.com/mafintosh/mud](http://github.com/mafintosh/mud) for (non-existing) documentation.

## Install

### Gem

    $ gem install mud

### Bundler

Add following line to your project's Gemfile

    gem 'mud'

## Dependencies

* hpricot 0.8.4
* sinatra 1.1.2
* thor 0.14.6
